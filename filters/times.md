---
title: times
---

Multiplies a number by another number.

```liquid
{% raw %}
{{ 3 | times: 2 }}
{% endraw %}
```

```text
{{ 3 | times: 2 }}
```

```liquid
{% raw %}
{{ 24 | times: 7 }}
{% endraw %}
```

```text
{{ 24 | times: 7 }}
```

```liquid
{% raw %}
{{ 183.357 | times: 12 }}
{% endraw %}
```

```text
{{ 183.357 | times: 12 }}
```
