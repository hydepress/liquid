---
title: downcase
---

Makes each character in a string lowercase. It has no effect on strings which are already all lowercase.

```liquid
{% raw %}
{{ "Parker Moore" | downcase }}
{% endraw %}
```

```text
{{ "Parker Moore" | downcase }}
```

```liquid
{% raw %}
{{ "apple" | downcase }}
{% endraw %}
```

```text
{{ "apple" | downcase }}
```
